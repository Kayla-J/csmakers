Algorithm Activity: Human Robot
Lesson description: This lesson includes students writing an algorithm to help a human robot accomplish a task. The robot must be given a set of step-by-step instructions, cannot create its own instructions, and cannot improvise.
Time required: 55 minutes
Alabama DLCS standards: 8th grade:
  • Computational thinker - Programming and Development #6
