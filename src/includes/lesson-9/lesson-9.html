Multiple Numeric Variables
Lesson description: All useful computer programs store and manipulate information. In order to store information that can be changed (i.e. vary), we need variables. This lesson involves the creation of a program that utilizes variables, (x,y) coordinates, and the modulo operator.
Time required: 65 minutes
Alabama DLCS standards: 8th grade:
